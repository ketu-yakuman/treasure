🀄 精算ルール説明
📌 基本
スタート点35,000点（3人合計 105,000点）
精算単位（持ち点 − 35,000）÷ 1,000 ＋ ウマ
表示精算点で表示（例：+95 = 95,000点受け取り）
🏆 ウマ（選択制）
なし加減点なし
5-101位+10 / 2位+5 / 3位-15
15-01位+15 / 2位 0 / 3位-15
10-201位+20 / 2位+10 / 3位-30
🐔 焼き鳥（選択制）
対象1度も和了せずに終局したプレイヤー
点数なし / 10,000 / 20,000 / 30,000点
支払い先焼き鳥でないプレイヤーへ均等に支払い
例）焼き鳥10,000点・Aが焼き鳥
→ AはBとCへ各5,000点支払い
※1位が焼き鳥でもペナルティあり
📉 沈みボーナス（ON/OFF）
条件原点（35,000点）未満で終局
ペナルティ-10,000点（該当者ごと）
受け取り減点分は全額1位が受け取る
例）A:47,000 / B:33,000 / C:25,000
B・Cが各-10,000 → Aが+20,000受け取り
🎯 オカ（選択制）
仕組み精算基準を40,000点にして生じる差額を上位者へ配分
1位+151位に+15加算
1位+10/2位+51位+10・2位+5加算
💥 飛ばしボーナス（ON/OFF）
条件0点以下になったプレイヤーが発生
東場飛ばされた → 飛ばした人へ 20,000点
南場以降飛ばされた → 飛ばした人へ 10,000点
ダブロン2人で飛ばした場合は頭割り
例）東場・Aが飛ばされ、BとCがダブロン
→ AはBとCへ各10,000点支払い
※ルールはゲーム前に参加者全員で確認してください
